# Capability status

## Portal foundation — available

This documentation website is a standalone static artifact with local search, locale
scaffolding, and a reviewable authoring workflow.

## Operator guide — pending verification

Task-oriented instructions for self-hosting, authentication, configuration, operations,
backup, recovery, and browser journeys will be published only after their runtime,
administration, and delivery contracts are accepted and retested.

## Deferred capabilities

MCP and evidence-aware chat are not available setup targets. They are planned
capabilities and will receive documentation only when their delivery contracts are
accepted.

## How to use this status

Do not infer availability from a backlog label, screenshot, or an unfinished source
module. An operator guide must be backed by current runtime behavior, validators, and
tests at the time it is published.
